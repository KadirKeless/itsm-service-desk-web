# ITSM Service Desk - Enterprise Web Version 🚀

This repository showcases the modernized, web-based enterprise backend of the IT Service Management (ITSM) application. It represents a complete **architectural migration** from a legacy Java desktop application (Swing) to a scalable, production-ready enterprise web API ecosystem.

## 🛠️ Modern Tech Stack
*   **Framework:** Spring Boot (Spring Web, Spring Data JPA)
*   **Language:** Java 17+
*   **Build & Dependency Tool:** Maven
*   **Database:** MySQL Database
*   **ORM / JPA Provider:** Hibernate

## ⚙️ Key Architectural Highlights
*   **Desktop-to-Web Migration:** Re-engineered the core Java business logic into a completely decoupled, stateless RESTful web service.
*   **Layered Architecture:** Implemented standard enterprise software patterns with strict separation of concerns across Controller, Service, and Repository layers.
*   **Data Modeling & ORM:** Designed robust entity relationships using Spring Data JPA and Hibernate to handle IT ticket workflows, user roles, and database persistence on MySQL.
*   **RESTful API Design:** Clean, structured endpoints designed to handle enterprise IT operations and service desk demands efficiently.
